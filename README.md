# PiAI
Bootstrap and manage an independent local agent via Github

## Prerequisites

1. A raspberry pi with Raspbian installed
     - connected to the Internet
2. Dedicated GMail account created
3. Dedicated Github account created using GMail from 2.
4. Dedicated Discord account created using GMail from 2.
5. One or both of:
     - Open AI ChatGPT Plus account
     - Anthropic Claude Plus account
6. CloudFlare account created using GMail from 2.
     - set up custom domain

## Bootstrapping

Run `...` on the pi

This will install the following dependencies:
- Docker Tools
- Github Tools
- Python
- NodeJS
- Codex CLI
- Claude CLI


1. Populate repository secrets for
    - Github PAT
    - Discord PAT
    - GMail username
    - GMail password
